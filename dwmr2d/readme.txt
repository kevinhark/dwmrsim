Included Files:
dwmr2d.m - Entry point for 2d diffusion simulation
se_diff.m - sample script that simulates a spin-echo experiment 
ste_diff.m - sample script that simulates a stimulated-echo experiment 
vetting.m - a WIP script which compares computational solutions with analytical solutions
dwmr2dp.cu - CUDA kernel, editing this will require recompiling
dwmr2dp.ptx - partially compiled CUDA kernel (aka GPU assembly), do not edit this
